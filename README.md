# Cookbook App

A personalized iOS cookbook app powered by Claude AI. Generates recipes based on your pantry, learns from your ratings, and helps you plan the week.

**Note:** The app source lives in the `CookbookApp/` folder. Move it to the `craigwmc2/Cookbook-app` repository.

## Features

- AI Recipe Generation via Claude API (with prompt caching)
- Smart rating system — low ratings teach the AI to avoid similar ingredients
- Pantry manager with expiration tracking and low-stock alerts
- Weekly meal planner (breakfast / lunch / dinner)
- Auto-generated shopping list from the weekly plan
- Full-screen Cooking Mode with per-step countdown timers
- Recipe scaling (servings stepper adjusts all ingredient amounts)
- Nutritional info per recipe
- Allergy exclusions: Crawfish, Shellfish, Peanuts, and more
- "Surprise Me" one-tap quick meal suggestion from pantry
